Remix OS for PC Beta上手体验（1）

1.前言——Remix OS for PC是什么

要了解这个问题，我们要先理解Remix OS是什么。我们先来看看Remix官网是怎么说的。

简单来说，就是一个类似Windows电脑操作系统的移动设备操作系统，说白了，就是深度定制Android。现在大家可能就明白了，Remix OS for PC其实就是可以在电脑上体验到这个类似Windows电脑操作系统的操作系统。

2.说明——笔者电脑配置说明

主要参数为

  • 处理器 英特尔 第二代酷睿 i3-2310M @ 2.10GHz 双核

  • 内存 6 GB ( 海力士 DDR3 1333MHz / 金士顿 DDR3 1600MHz )

  • 主硬盘 日立 HTS725050A7E630 ( 500 GB / 7200 转/分 )

  • 主显卡 英特尔 HD Graphics Family ( 华硕 )

3.安装

整个安装过程非常简单。首先下载Remix OS for PC Beta压缩包，解压后有两个重要文件，一个是Remix OS for PC Beta的镜像文件，一个是Remix OS for PC 的安装工具，即U盘启动盘制作工具。

然后使用Remix OS for PC的U盘启动盘制作工具烧录U盘。本文使用的是USB3.0的16g台电U盘。

制作完成后会提示需要重新启动。点击确定后，系统重新启动。本机是华硕电脑，按ESC键后进入启动项选择。找到U盘启动后，按enter键进入。接下来几乎一路按enter键就好了，比windows安装还要简单。

4.启动

本机支持UEFI启动，且优先UEFI启动。在启动过程中，没有发现明显的问题。启动过程也是相当快的，虽没有精准计算过，但不超过3分钟，并没有出现一些网友反馈的要5分钟之久。

Remix OS 有两个模式选择，一个是Resident Mode（普通模式），一个是Guest Mode（访客模式）。默认第一个普通模式。大家可以按自己的需求自由选择。笔者选择的是普通模式。

5.初步使用

  • 开始菜单

这个功能跟安卓原生系统中的应用抽屉差不多，都是一个类似的应用集合的地方，当然Remix对其进行了改进。对于一个电脑操作系统来说，这个功能虽然有模仿windows的地方，但也符合大多数人的使用习惯。对比Android系统来说，这确实是很让人耳目一新的，期待Remix在这点能有更大的创新。

其实，笔者认为，这个开始菜单跟chrome的启动器更像一点。

开始菜单中，可以搜索软件，也可以对应用排序。排序方式主要有三种，名称，更新时间和使用频率。

同时，也可以把软件锁定到任务栏中，也可以卸载和查看软件详情。

当然，也可以点击电源图标，进行关机，待机和重新操作。

对比windows，少了注销。这里笔者认为是注销对于Android系统来说，确实没什么意义，所以Remix没有加入这个功能。

  • 状态栏&底部虚拟按键

下图就是Remix OS for PC的状态栏。仔细观察上图的状态栏，可以发现其中有Android底部虚拟按键和window操作系统的影子。我觉得更像是两者的结合体。

对比安卓虚拟按键，可以看到是都有返回上一页面和回到主页面，不同的是多任务按键没有了。其实也不能说没有了，这时候的多任务切换被任务栏替代了，类似window。可以说是这个状态栏已经包含了安卓的虚拟按键。

对比window状态栏，最大的改变就是多了返回。其他的通知，日期与时间，网络连接，电池状态，输入法等都一应俱全。

然而在实际使用中，由于软件坐上角都有返回的三角形返回键，所以状态栏的返回键使用很少。个人认为，可以给返回键及主页面键配上快捷键，这样子的话对于键盘操作来说就比较方便。

  • 通知管理

点击状态栏的最右边按钮，即可打开通知中心。Remix的通知管理做了分类管理，整个通知特面比较整洁。同时，通知中心也集成了一些常用功能。如：旋转，收起，截图和设置。其中截图功能是我用的最多的地方，本文中的所有图片，全部来自于其截图，由使用Prt Sc键唤出，也可点击通知中心底部的第三个按钮。

  • 日期与时间

日期有一点不太符合中国人的习惯，就是没有农历。

  • 网络连接

紧接着，我没有拔掉网线，而是直接打开WiFi。这个时候，神奇的事情发生了，WiFi打不开了。

我打开WiFi设置界面，猛按了好几十下都没有反应。
